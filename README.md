# DISHA – Dropout Insight & Support for Holistic Academics 🎓✨

> **An AI-powered multi-university ecosystem to reduce student dropouts by 90–95%.**  
> Built for Boards • Universities • Teachers • Students

---

## 🌐 Overview

**DISHA** is an **AI-driven centralized platform** designed to identify, analyze, and resolve student issues across multiple universities, boards, and institutions.  
The goal is to **reduce academic dropouts** by predicting risks early, alerting stakeholders, and providing **personalized counselling + solutions**.

---

## 🚩 Key Features

- 🔹 **Multi-Level Dashboards**
  - **Board Level** → For SPPU, AKTU, CBSE, etc.
  - **University Level** → For MIT ADT, Galgotias, etc.
  - **Teacher Level** → Teacher-specific insights
  - **Student Level** → Personalized student reports & counselling

- 🔹 **AI Predictor Models**
  - Student Risk Prediction (Dropout Likelihood)
  - Sentiment & Behavior Analysis (Text, Voice, Feedback)
  - Financial Stress Detection
  - Academic Performance Forecasting
  - Attendance & Engagement Tracking
  - Entertainment vs Genuine Distress Detection

- 🔹 **Centralized Data Sources**
  - 🎓 Academic Records
  - 📊 Attendance
  - 💰 Financial Background
  - 🧠 Behavioral & Sentiment Analysis
  - ❤️ Health & Well-being
  - 📱 Student Feedback & Digital Footprints

- 🔹 **Smart Alerts & Counselling**
  - Automated alerts to **Teachers & Universities**
  - **Personalized counselling plans** generated by AI
  - Early intervention for **financial aid, mentoring, or mental health support**

---

## 🛠 Tech Stack

- **Frontend** → Next.js / React.js + Tailwind CSS  
- **Animations** → Framer Motion + GSAP  
- **Backend** → Node.js + Express.js  
- **Database** → MongoDB + PostgreSQL (Hybrid)  
- **AI Models** → 
  - Classification Models (Dropout Risk)
  - NLP (Sentiment & Feedback Analysis)
  - Predictive Models (Academic & Financial Forecasts)  
- **Deployment** → Docker + Vercel/Render
